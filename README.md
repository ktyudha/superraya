# SUPERRAYA

Superraya adalah sebuah Web Company Profile. Proyek ini dibangun menggunakan Laravel 11.

## Persyaratan

-   PHP 8.1 atau lebih tinggi
-   Composer
-   Laravel 11
-   Database (MySQL, PostgreSQL, dll.)

## Instalasi

**Clone Repository**

```bash
git clone https://github.com/ktyudha/superraya.git
```
